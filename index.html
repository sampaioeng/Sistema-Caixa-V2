5 — Atividades

Importe os códigos, valores e prazos a partir de uma planilha, colando do edital, ou adicione manualmente. Pode pular este passo e importar depois.

CódigoValor OS (R$)Prazo (dias)
Formato esperado (ponto-e-vírgula ou vírgula como separador):
CÓDIGO ; VALOR ; PRAZO — Ex: G401;650,00;30
ou CÓDIGO ; DESCRIÇÃO ; VALOR ; PRAZO — Ex: G401;Avaliação Residencial;650,00;30
A primeira linha pode ser cabeçalho (será ignorada automaticamente).
Cole abaixo o texto do edital com as atividades. Formatos aceitos:
G401 - Avaliação Residencial - R$ 650,00 - 30 dias
G401 Avaliação Residencial R$ 650,00 30
G401;650,00;30

Formato esperado:
MUNICÍPIO ; VALOR_DESLOC_NORMAL ; POLO — Ex: Belo Horizonte;0;BH
Adicione ;sim ao final para marcar como especial — o valor será calculado como 21% do normal.
Ex: Santa Luzia;45;BH;sim → especial = R$ 9,45
